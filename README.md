# Requirements!
  - **npm - v6.14.6**
  - **Node JS - v12.18.3**
  

### Installation

Clone the repo:
```sh
$ git clone https://github.com/einerb/prometheus-lab.git
```
Access the project:
```sh
$ cd prometheus-lab
```
Install the packages:
```sh
$ npm i
```
Run the server:
```sh
$ node index.js
```
Access the url from the browser:
```sh
localhost:8080/metrics
```
